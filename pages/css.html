CSS

Басты бетке оралу

1. CSS деген не? Қалай қосамыз?

CSS (Cascading Style Sheets) — HTML-ді әдемі етіп бояйтын сиқыр!

<!-- 1. Inline (жаман тәсіл) --> <p style="color: red; font-size: 20px;">Мәтін</p> <!-- 2. Internal (бір бетке) --> <style> p { color: blue; } </style> <!-- 3. External (ең дұрыс!) --> <link rel="stylesheet" href="style.css">

2. Негізгі селекторлар

* { margin: 0; padding: 0; } /* барлығы */ p { color: white; } /* тег бойынша */ .title { font-size: 32px; } /* класс */ #header { background: #1e293b; } /* id */ p.highlight { color: #58a6ff; } /* класс + тег */ a:hover { color: #f472b6; } /* ховер */

3. Түстер, шрифт, фон

color: #58a6ff; /* мәтін түсі */ background: #0f172a; background: linear-gradient(135deg, #1e293b, #0f172a); font-family: 'Comic Relief', cursive; font-size: 18px; font-weight: 700; text-align: center; text-shadow: 0 2px 4px rgba(0,0,0,0.5);

4. Box Model – әр элементтің құрылымы

margin → padding → border → content
.box { width: 300px; height: 200px; padding: 20px; border: 5px solid #58a6ff; margin: 30px auto; box-sizing: border-box; /* маңызды! */ }

5. Flexbox – заманауи орналастыру

.container { display: flex; justify-content: center; /* горизонталь */ align-items: center; /* вертикаль */ gap: 20px; flex-wrap: wrap; } .item { flex: 1; } /* тең бөлу */

6. Grid – кесте сияқты, бірақ күштірек

.grid { display: grid; grid-template-columns: 1fr 1fr 1fr; /* 3 баған */ grid-template-rows: 100px 200px; gap: 16px; } .grid-item:nth-child(1) { grid-column: 1 / 4; /* бүкіл жолды алады */ }

7. Responsive дизайн (мобильдіге бейімдеу)

@media (max-width: 768px) { .container { flex-direction: column; } h1 { font-size: 28px; } } /* Mobile first */ .container { width: 100%; padding: 10px; } @media (min-width: 1024px) { .container { max-width: 1200px; margin: 0 auto; } }

8. Анимация және трансформация

.card:hover { transform: translateY(-10px) scale(1.05); transition: all 0.4s ease; } @keyframes pulse { 0% { transform: scale(1); } 50% { transform: scale(1.1); } 100% { transform: scale(1); } } .pulse { animation: pulse 2s infinite; }

9. CSS Variables (өте пайдалы!)

:root { --main-color: #58a6ff; --bg-dark: #0f172a; --radius: 16px; } .card { background: var(--bg-dark); color: var(--main-color); border-radius: var(--radius); }

10. Senior деңгейі – нағыз профи трюктар

/* Clamp – шрифт өлшемі автоматты өзгереді */ h1 { font-size: clamp(24px, 5vw, 60px); } /* Container queries (жаңа!) */ .container { container-type: inline-size; } @container (max-width: 500px) { .card { flex-direction: column; } } /* Glassmorphism */ .glass { background: rgba(255, 255, 255, 0.1); backdrop-filter: blur(12px); border: 1px solid rgba(255, 255, 255, 0.2); } /* Scroll snap */ .scroll-container { scroll-snap-type: y mandatory; overflow-y: scroll; height: 100vh; } .section { scroll-snap-align: start; }

Практика: әдемі карточка жасайық!

HTML:
<div class="modern-card"> <h3>LearnProgLang</h3> <p>Қазақша программинг</p> </div>
CSS:
.modern-card { padding: 30px; background: linear-gradient(135deg, #1e293b, #0f172a); border-radius: 20px; text-align: center; box-shadow: 0 10px 30px rgba(0,0,0,0.5); transition: 0.4s; border: 1px solid #334155; } .modern-card:hover { transform: translateY(-12px); box-shadow: 0 20px 40px rgba(88,166,255,0.3); border-color: #58a6ff; }
