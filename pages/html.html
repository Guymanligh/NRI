HTML

Басты бетке оралу

1. HTML деген не?

HTML (HyperText Markup Language) — веб-беттің қаңқасы. Барлық сайт осыдан басталады!

<!DOCTYPE html> <html lang="kk"> <head> <title>Менің бірінші сайтым</title> </head> <body> <h1>Сәлем, әлем!</h1> </body> </html>

2. Негізгі тегтер

<h1>...<h6> # тақырыптар
<p>Параграф</p>
<a href="https://google.kz">Google-ға сілтеме</a>
<img src="photo.jpg" alt="Суреттің сипаттамасы">
<strong>Қою</strong> және <em>егер</em>
<br> # жол үзу
<hr> # сызық

3. Тізімдер

# Реттелген (нөмірлі)
<ol> <li>Бірінші</li> <li>Екінші</li> </ol>
# Реттелмеген (нүктелі)
<ul> <li>Алма</li> <li>Банан</li> </ul>

4. Кесте (table)

<table border="1"> <tr> <th>Аты</th> <th>Жасы</th> </tr> <tr> <td>Азамат</td> <td>25</td> </tr> </table>

5. Формалар – пайдаланушыдан мәлімет алу

<form action="/submit" method="POST"> <label>Атыңыз:</label><br> <input type="text" name="name" placeholder="Аты-жөніңіз" required><br><br> <label>Пароль:</label><br> <input type="password" name="pass"><br><br> <input type="checkbox" name="agree"> Мен келісемін<br><br> <input type="submit" value="Жіберу"> </form>

6. Семантикалық HTML5 тегтері (өте маңызды!)

<header> # сайттың жоғарғы бөлімі
<nav> # навигация
<main> # негізгі мазмұн
<section> # бөлім
<article> # мақала
<aside> # бүйірлік панель
<footer> # төменгі бөлім

7. Медиа: сурет, видео, аудио

<img src="cat.jpg" alt="Мысық" width="500">
<video controls width="600"> <source src="video.mp4" type="video/mp4"> Браузеріңіз видео қолдамайды </video>
<audio controls> <source src="music.mp3" type="audio/mpeg"> </audio>

8. Meta тегтер (SEO үшін өте маңызды)

<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta name="description" content="Қазақша HTML оқу курсы"> <meta name="keywords" content="html, қазақша, оқыту"> <title>LearnProgLang - HTML</title> </head>

9. Практика: толық сайт қаңқасы

<!DOCTYPE html> <html lang="kk"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Менің портфолиом</title> </head> <body> <header> <h1>Азамат Бағдатов</h1> <nav> <a href="#about">Мен туралы</a> <a href="#projects">Жобалар</a> </nav> </header> <main> <section id="about"> <h2>Мен туралы</h2> <p>Мен веб-әзірлеушімін...</p> </section> </main> <footer> <p>© 2025. Барлық құқықтар қорғалған.</p> </footer> </body> </html>

10. Senior деңгейі – заманауи HTML трюктары

# Деректерді сақтау (data-атрибуттар)
<div data-user-id="123" data-role="admin">Азамат</div>
# ARIA (мүмкіндігі шектеулі адамдарға)
<button aria-label="Жабу" aria-expanded="false">✕</button>
# Microdata (іздеу жүйелеріне арналған)
<div itemscope itemtype="https://schema.org/Person"> <span itemprop="name">Азамат</span> </div>
