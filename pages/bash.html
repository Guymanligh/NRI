Bash

Басты бетке оралу

1. Bash деген не? Неге керек?

Bash — Linux/macOS-тың командалық қабығы. GUI-сыз серверді басқару, скрипт жазу, автоматтандыру — бәрі осы арқылы жүреді.

Нағыз программисттің суперқабілеті!

2. Терминалды қалай ашамыз?

# Windows → Git Bash, WSL2 немесе Windows Terminal
# macOS → Cmd + Space → "Terminal"
# Linux → Ctrl + Alt + T

3. Негізгі командалар (күнде 100% қолданасың)

pwd # қай жердемін?
ls # файлдарды көрсету
ls -la # жасырын файлдармен бірге
cd Documents # папкаға кіру
cd .. # бір деңгей жоғары
cd ~ # үй папкасына
cd / # түбір папка
mkdir project # жаңа папка
touch file.txt # бос файл құру
rm file.txt # файлды өшіру
rm -r folder # папканы өшіру (абайла!)
cp file.txt copy.txt # көшірме жасау
mv old.txt new.txt # атын өзгерту / жылжыту
cat file.txt # файл мазмұнын шығару
less file.txt # үлкен файлды парақтап оқу
echo "Сәлем!" # мәтін шығару
echo "Сәлем!" > file.txt # файлға жазу
echo "Қосымша" >> file.txt # қосу

4. Файлдармен жұмыс (пайдалы трюктар)

find . -name "*.js" # барлық JS файлдарды табу
grep "error" log.txt # мәтіннен іздеу
grep -r "TODO" . # бүкіл проектіден іздеу
wc -l file.txt # жолдар санын санау
head -n 10 file.txt # алғашқы 10 жол
tail -n 10 file.txt # соңғы 10 жол
tail -f log.txt # логты "тірі" бақылау

5. Процестер және жүйе

ps aux # барлық процестер
top # тікелей мониторинг (q басып шығу)
htop # әдемі top (орнату керек)
kill 12345 # процесті тоқтату
kill -9 12345 # күштеп өлтіру
df -h # диск орны
du -sh * # папкалар өлшемі
free -h # жедел жады

6. Бірінші скрипт жазамыз!

nano hello.sh

Ішіне жаз:

#!/bin/bash echo "Сәлем, әлем!" echo "Бүгін: $(date)" echo "Сен: $USER"
chmod +x hello.sh # орындауға рұқсат
./hello.sh # іске қосу

7. Айнымалылар (Variables)

echo 'Сәлем, $NAME' # бір жақшамен өзгермейді
readonly PI=3.14 # өзгертуге болмайтын
export DATABASE_URL="postgresql://..." # басқа скрипттерге беру

8. Шарттар (if, else)

if [ "$1" = "admin" ]; then echo "Қош келдіңіз, әкімші!" elif [ "$1" = "user" ]; then echo "Қалыпты пайдаланушы" else echo "Кімсің сен?" fi

9. Циклдар (for, while)

# 1-ден 10-ға дейін
for i in {1..10}; do echo "Сан: $i" done
# Барлық .txt файлдарды өңдеу
for file in *.txt; do echo "Өңдеп жатырмын: $file" done

10. Функциялар

11. Пайдалы трюктар (Middle+)

history | grep git # бұрынғы командалардан іздеу
Ctrl + R # кері іздеу (сиқыр!)
!! # соңғы команданы қайталау
sudo !! # соңғы команданы sudo-мен қайталау
cd - # алдыңғы папкаға қайту
alias gs='git status' # қысқарту (~/.bashrc-қа жаз)

12. Senior деңгейі – нағыз DevOps

# Бір жолда бірнеше команда
cd /tmp && wget file.zip && unzip file.zip && rm file.zip
# && = алдыңғысы сәтті болса ғана
# || = алдыңғысы сәтсіз болса
make || echo "Қате шықты!"
# Пайп (pipe) – бір команданың шығысын екіншісіне беру
cat log.txt | grep "ERROR" | wc -l
# Here-document
cat << EOF > config.txt host=localhost port=5432 user=admin EOF
