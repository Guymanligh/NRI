Git

Басты бетке оралу

1. Git деген не? Неге керек?

Git – кодтың әр өзгерісін сақтайтын, уақытқа саяхат жасай алатын сиқырлы құрал.

Git-сіз өмір: index.html → index_final.html → index_really_final.html

Git-пен өмір: 10 секундта кешегі, тіпті өткен аптадағы нұсқаға қайтасың!

Git ≠ GitHub! Git – программа, GitHub – бұлттағы сервер.

2. Git орнату және алғашқы баптау

Орнату

Windows

winget install --id Git.Git -e --source winget

MacOS

brew install git

Linux

apt-get install git-core
yum install git-core
zypper in git-core

Алғашқы баптау

git --version
git config --global user.name "Сенің атың"
git config --global user.email "email@example.com"
git config --global core.editor "code --wait"
git config --global init.defaultBranch main

3. Алғашқы репозиторий және негізгі командалар

mkdir my-project && cd my-project
git init
git status
git add .
git commit -m "алғашқы коммит"
git log --oneline

.gitignore файлы (мүлдем керек!)

# .gitignore ішіне мынаны жаз node_modules/ .env *.log .DS_Store dist/ build/

4. Уақыт сапары – қателерді қайтару

git revert HEAD # қауіпсіз қайтару
git reset --hard HEAD~3 # соңғы 3 коммит жоғалады!
git reflog # барлық әрекеттер тарихы
git reset --hard HEAD@{1} # "өткенге оралу"

5. Branch – Git-тің жүрегі

git switch -c feature/login
git switch main
git merge feature/login
git branch -d feature/login

Merge conflict шықса:

<<<<<<< HEAD Ескі мәтін ======= Жаңа мәтін >>>>>>> feature/login

6. GitHub және Remote репозиторий

git remote add origin https://github.com/username/repo.git
git branch -M main
git push -u origin main
git pull
git push

SSH кілт (пароль сұрамайтын болу үшін)

ssh-keygen -t ed25519 -C "email@example.com"

7. Git Rebase – таза әрі әдемі тарих

git switch feature/payment
git rebase main
git rebase -i HEAD~5 # squash, fixup жасаймыз

8. Git Flow vs GitHub Flow

GitHub Flow (қазіргі тренд):
Тек main + әр функцияға Pull Request

Git Flow: main + develop + release + hotfix

9. Pull Request және Code Review

Жақсы PR шаблоны:

# Тақырып [FEATURE] Пайдаланушы тіркеу формасы # Не істедім? - Email валидация қостым - Тестілер жаздым # Скриншот ![screenshot.png] # Чеклист - [x] Тестілер өтті - [x] ESLint таза

10. Senior деңгейі – профи болу

# Pre-commit хук (husky) npx husky add .husky/pre-commit "npm test && npm run lint"
# Conventional Commits feat: жаңа кнопка қосу fix: логин багын түзету docs: README жаңарту chore: dependency жаңарту
# Үлкен файлдар (Git LFS) git lfs install git lfs track "*.psd" "*.mp4"
# Бірнеше тармақта бір уақытта жұмыс (worktree) git worktree add ../myproject-hotfix hotfix/bug123
