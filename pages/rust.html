1. Rust деген не?

Ең қауіпсіз, ең жылдам, memory-safe тіл. Linux kernel, Deno, Firefox бөліктері Rust-та.

2. Орнату

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
cargo new hello_rust
cd hello_rust && cargo run

3. Негізгі синтаксис

fn main() {
let name = "Азамат";
let mut age = 27;
age += 1;
println!("Сәлем, {name}! Жасың: {age}");
}

// Ownership
let s1 = String::from("сәлем");
let s2 = s1; // s1 енді жарамсыз!

4. async/await

#[tokio::main]
async fn main() {
let body = reqwest::get("https://httpbin.org/get").await.unwrap().text().await.unwrap();
println!("{}", body);
}
