1. Java деген не? Неге әлі №1?

Java – «бір рет жазсаң – кез келген жерде іске қосылады» (Write Once, Run Anywhere). Android қосымшалары, банк жүйелері, үлкен корпорациялар (Google, Yandex, Kaspi, Halyk), микросервистер (Spring Boot), ойындар (Minecraft) – бәрі Java-да!

4. Шарттар және циклдар

if (age >= 18) { System.out.println("Ересек"); } else { System.out.println("Бала"); } for (int i = 0; i < 5; i++) { System.out.println(i); } for (String city : cities) { System.out.println(city); } int i = 0; while (i < 3) { System.out.println("Сәлем " + i); i++; }

5. Класс және объект (OOP негізі)

public class Car { private String brand; private int year; public Car(String brand, int year) { this.brand = brand; this.year = year; } public void start() { System.out.println(brand + " қозғалтқышы іске қосылды!"); } public String getBrand() { return brand; } } Car myCar = new Car("Toyota", 2024); myCar.start();

6. Record – қысқа деректер класы (Java 16+)

public record Person(String name, int age) { } var p1 = new Person("Али", 25); var p2 = new Person("Али", 25); System.out.println(p1.equals(p2)); // true

7. Коллекциялар (List, Set, Map)

import java.util.*; List fruits = new ArrayList<>(); fruits.add("алма"); fruits.add("банан"); Set unique = new HashSet<>(); Map ages = new HashMap<>(); ages.put("Азамат", 27); fruits.forEach(System.out::println);

8. Stream API – сиқырлы сүзгі (Java 8+)

List numbers = List.of(1, 5, 2, 8, 3); numbers.stream() .filter(n -> n > 3) .sorted() .map(n -> n * 2) .forEach(System.out::println); // 10, 16
