Басты бетке оралу

1. C++ деген не? Неге үйренеміз?

C++ – C тілінің заманауи, объектіге бағытталған нұсқасы. Ойындар (Unreal Engine), браузерлер (Chrome), серверлер (Yandex, VK), графика, робототехника – бәрі C++-та!

2. Орнату және алғашқы программа

# Linux sudo apt install g++ build-essential # macOS brew install gcc # Windows → MSYS2, MinGW-w64 немесе Visual Studio

3. Негізгі жаңалықтар C-тан айырмашылығы

#include // C-тағы printf орнына using namespace std; // std:: деп жазбай-ақ қоюға болады cout << "Сәлем"; // шығару cin >> x; // кіріс string name = "Али"; // мәтін түрі (C-та жоқ!) bool isTrue = true;

4. Класс және объект (OOP негізі)

class Car { public: string brand; int year; void start() { cout << brand << " қозғалтқышы іске қосылды!\n"; } }; int main() { Car myCar; myCar.brand = "Toyota"; myCar.year = 2023; myCar.start(); }

5. Конструктор және деструктор

class Student { public: string name; int age; // Конструктор Student(string n, int a) { name = n; age = a; cout << name << " құрылды!\n"; } // Деструктор ~Student() { cout << name << " жойылды!\n"; } };

6. Инкапсуляция (private/public)

class BankAccount { private: double balance = 0; public: void deposit(double amount) { if (amount > 0) balance += amount; } double getBalance() { return balance; } };

7. Мұрагерлік (Inheritance)

class Animal { public: void eat() { cout << "Ням-ням\n"; } }; class Dog : public Animal { public: void bark() { cout << "Үр-үр!\n"; } }; int main() { Dog rex; rex.eat(); // мұраға алынған rex.bark(); }

8. Полиморфизм және виртуалды функциялар

class Shape { public: virtual void draw() { cout << "Фигура сызылды\n"; } }; class Circle : public Shape { public: void draw() override { cout << "Дөңгелек сызылды\n"; } }; void render(Shape* s) { s->draw(); } // полиморфизм
